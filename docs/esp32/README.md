# ESP32系列

ESP32系列是乐鑫推出的一系列功能强大的Wi-Fi和蓝牙模块，提供高性能处理器、丰富的外设和广泛的应用支持，适用于物联网和嵌入式应用。

## 产品列表

### ESP32-S3

### ESP32-C3

### ESP32-C6

### ESP32-S2

### ESP32-S3 (RISC-V)

- Xtensa LX7 双核处理器
- 2.4 GHz Wi-Fi
- Bluetooth 5 (LE)
- 丰富的GPIO接口
- 支持摄像头接口
- 支持显示接口

### ESP32-C3 (RISC-V)

- RISC-V 单核处理器
- 2.4 GHz Wi-Fi
- Bluetooth 5 (LE)
- 低功耗设计
- 紧凑型封装

### ESP32-C6

- RISC-V 多核处理器
- Wi-Fi 6
- Bluetooth 5 (LE)
- Thread/Zigbee 支持
- 低功耗物联网应用

### ESP32-S2

- Xtensa LX7 单核处理器
- 2.4 GHz Wi-Fi
- 丰富的USB接口
- 适用于物联网应用
