Appinventor环境搭建

发表于 1970-01-01 更新于 2020-07-24

最近在少儿机器人机构中开了一个强化班教孩子们全流程制作一个OTTO机器人并给机器人编程调试。

  • 建模软件采用FreeCAD。
  • 切片软件采用 Cura 。
  • 3D 打印机采用。
  • 编程软件采用[米思齐](http://www.mixgo.org/work?board=Arduino/Genuino Uno)。
  • 遥控OTTO机器人可以采用官方APP，但是因为我们是全流程，所以咱们要用Appinventor来制作一款APP。

Appinventor 官方的编程环境打不开，国内有几个本地环境更新不及时，自己学生的账户不好掌控，那么我们很有必要建立一个本地的局域网服务器。

  • 下载Appinventor源代码,并进入代码目录。

  • 安装 vagrant 虚拟机管理软件,并初始化虚拟机。

  • 本地ssh登陆虚拟机。

  • 在虚拟机上一键构建。

  • 在虚拟机上启动Appinventor服务器服务器。

Sratch课程公告

发表于 1970-01-01 更新于 2020-07-13

[TOC]

熊猫散步

  • 认识慧编程。

  • 认识运动，控制,外观模块。

  • 学习控制模块中的重复执行逻辑。

  • 重难点–设置熊猫的旋转方式。

活泼的Crab

  • 认识外观，声音模块。
  • 学会添加角色的造型。
  • 简单认识随机数。
  • 重难点–用程序控制角色造型的切换。

舞会

  • 控制模块中时间的作用。
  • 多个造型重复切换的编程逻辑。
  • 背景重复切换的编程逻辑。
  • 区分声音模块中播放声音与播放声音等待播放完毕两个指令。
  • 重难点–对两个角色分别编程。

Pico秀

  • 如何用程序控制角色的朝向。
  • 认识并运用外观模块中的“说”指令。
  • 区分程序的顺序执行与重复执行。
  • 重难点–理解程序块，学习程序块的定义。

贺卡

  • 背景的意义与切换
  • “如果…那么…”控制逻辑的学习。
  • 外观模块中显示与隐藏的意义与作用。
  • 运动模块中“移到…”与“在1秒内滑行到的区别”。
  • 重难点–多角色的程序组合。

龙珠

  • 认识运动模块中的移到随机位置指令。
  • 认识运动模块中“面向”与“将旋转方式设为左右翻转”混合编程。
  • 学习Scratch 编写故事的方法与逻辑。

动画主题复习

  • 用键盘控制角色运动。
  • 用键盘控制角色跳动。
  • 用键盘控制角色执行一系列动作。
  • 重难点–修改素材，制作动画。

乒乓球赛

  • 认识程序的并行执行

  • 认识变量并掌握变量的创建方法。

  • 学会使用侦测模块的“鼠标的X坐标模块”。

  • 重难点–控制模块中的“重复执行“与“如果…那么…“的混合编程。

装扮特拉

  • 移到与在1秒内移到的区别。
  • 舞台区角色的图层。
  • 角色的位置与其程序坐标的对应关系。
  • 重难点–角色位置与角色的角色图层配合。
